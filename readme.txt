Launched in March 2021, blockchain Symbol makes it easy for anyone to issue transactions via a WordPress plugin. The power of blockchain for all. The development journey has just begun. We look forward to your support.

== Description ==

This plugin allows you to easily and securely issue transactions against the blockchain Symbol. No private keys are stored on the Wordpress side. 

Required

- Google Chrome
- SSS Extention
